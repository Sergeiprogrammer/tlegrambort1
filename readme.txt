#важно!
это бот для генерации картинок на основе Fusion Brain который будет оправлять их пользователю в телеграмм

#нужно для работы бота!
установить telebot,googletrans на сервер
pip install telebot
pip install googletrans==4.0.0-rc1
ввести свой токен в файле info.txt получив его у BotFather 
и ввести своb токены в файле telegram.py получив на сайте https://fusionbrain.ai/keys

#архитекура
этот бот получает запрос от пользователя на выполнение команды импортирует класс из файла main.py затем как аргумент передаёт его после чего отправляет его на сервер fusionbrain и получает результат в формате файла Base64 портирует его в png и отдаёт пользователю реализована систем что человек не может генерировать сообщение пока его имя есть в очереди на генерацию также хочу в будущих обновлениях реализовать лимиты на сообщение от конкретного человека!
