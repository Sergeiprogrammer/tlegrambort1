#Важно!
Это бот для генерации картинок на основе Fusion Brain, который отправляет их пользователю в Telegram.

#Что нужно для работы бота!:
Установить библиотеки telebot и googletrans на сервер:

вот один из вариантов:
pip install telebot
pip install googletrans==4.0.0-rc1

также нужно Ввести свой токен в файле info.txt, получив его у BotFather.
И ввести свои токены в файле telegram.py, получив их на сайте Fusion Brain.

Архитектура
Этот бот получает запрос от пользователя на выполнение команды, импортирует класс из файла main.py, затем передаёт его как аргумент. После этого запрос отправляется на сервер Fusion Brain и получает результат в формате Base64. Затем он конвертирует этот результат в PNG, загружает в папку Images и отправляет пользователю.

Реализована система, при которой пользователь не может генерировать сообщение, пока его имя есть в очереди на генерацию. Также в будущих обновлениях планируется реализовать лимиты на сообщения от конкретного пользователя. Фото пользователей не удаляются, а при новой генерации просто перезаписываются, так как Telegram не позволяет удалять их.
